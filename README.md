# 4-PJT

Création d'une blockchain, avec un wallet en ligne de commande et une interface web permettant de connaitre l'état de la blockchain.

## Utilisation
### Compilation
```shell
./compile
```
### Lancement d'un noeud
```shell
./start-node
```
### Interaction avec la blockchain
```shell
./wallet
```
